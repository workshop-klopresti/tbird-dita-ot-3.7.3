Tools

Invest in a good set of tools for doing all kinds of tasks around the house.

Useful tools include the following items:

  • Hammer
  • Screw driver set
  • Wrench set
  • Nails and screws
  • Level
  • Saws
  • Drill
  • Air pressure guage
  • Spade
  • Rake

Keep your tools organized in a tool box.
